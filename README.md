🚀 LangChain AI Playground (Gemini + Hugging Face + RAG)

A hands-on project demonstrating how to build AI-powered applications using LangChain, integrating chat models (Gemini).
Hugging Face open-source models, Chains, Runnables, Prompts, and basic RAG (Retrieval-Augmented Generation).


📌 Overview

This project explores the core building blocks of LangChain to create intelligent, modular, and scalable AI workflows. 
It focuses on combining LLMs with structured pipelines and external knowledge sources.

✨ Features
🤖 Chat Models Integration
Google Gemini (via LangChain)
Conversational AI pipelines
🤗 Hugging Face Models
Open-source LLM usage
Local/remote inference support
🔗 Chains
Sequential and logical flow of operations
Multi-step processing with LLMs
⚡ Runnables
Modern LangChain abstraction
Composable and reusable pipelines
🧠 Prompt Engineering
Prompt templates
Dynamic input handling
📚 Basic RAG (Retrieval-Augmented Generation)
Document loading
Embeddings + vector store
Context-based answering

🛠️ Tech Stack
Python
LangChain
Google Gemini API
Hugging Face Transformers
Vector Stores (FAISS / Chroma)
